# tic-tac-toe
# 🤖 智慧九宮格遊戲 (Tic-Tac-Toe AI)

這是一個基於 HTML/CSS/JavaScript 開發的經典九宮格遊戲，特色在於其**「困難模式」**採用了經典的 **Minimax 演算法**，讓電腦成為一個幾乎不可能擊敗的對手。

---

## ✨ 核心特色

* **雙模式體驗：**
    * **簡單模式 (Easy):** 電腦會隨機選擇空位下棋，適合放鬆或讓初學者練習。
    * **困難模式 (Hard):** 電腦將使用 Minimax 演算法，保證能達到最佳決策。想贏？您可能需要一點運氣或戰術的突破！
* **即時計分板：** 追蹤玩家、電腦和平局的戰績。
* **視覺回饋：** 勝利或平局時，格子會高亮顯示，並提供清晰的狀態訊息。

## 🧠 AI 核心：Minimax 演算法

困難模式下的電腦並不是亂猜的，它基於經典的 Minimax 搜尋演算法運行：

* **目標：** 在所有可能的走步中，找到對自己最有利（分數最高）且對對手最不利（分數最低）的那一步。
* **評估標準：**
    * AI 獲勝：給予高分（$10 - \text{深度}$）。
    * 玩家獲勝：給予低分（$\text{深度} - 10$）。
    * 平局：$0$ 分。
* **決策過程：** Minimax 會「預見」未來所有可能的棋局直到遊戲結束，透過遞迴計算來決定當前哪一步是最佳選擇。這確保了 AI 在不犯錯的情況下，**永遠不會輸**。

    直接在瀏覽器中開啟 `index.html` 文件即可開始遊戲。
